This project is a financial analytics tool designed to process Excel datasets and generate actionable business insights.

## Key Technical Features

- Built data processing pipelines using Pandas to compute revenue, tips, and performance metrics.
- Implemented dynamic filtering by date and staff to analyze financial trends.
- Designed an interactive dashboard for real-time business insights.
- Handles structured Excel input and transforms it into usable analytics.

## Impact

- Enabled detailed tracking of barber earnings and tips.
- Simplified financial analysis for business decision-making.
- Reduced manual effort required to process financial data.
