## Overview

This project focuses on building anomaly detection systems for industrial environments, specifically hydraulic systems and wind turbines. The goal was to identify abnormal behavior in time-series sensor data and improve system monitoring and reliability.

## Key Technical Work

- Developed anomaly detection models using LSTM and Transformer-based architectures (TranAD)
- Applied physics-informed approaches to better model system behavior and detect anomalies
- Processed and structured time-series sensor data for model training and evaluation
- Built a Streamlit dashboard for real-time anomaly visualization

## Wind Turbine System

- Implemented TranAD (Transformer-based anomaly detection model)
- Combined reconstruction and prediction errors to generate anomaly scores
- Tuned thresholds based on normal data distribution for accurate detection
- Achieved high detection performance with strong recall and minimal false negatives

## Hydraulic System

- Applied anomaly detection techniques on multi-sensor industrial data
- Designed preprocessing pipelines for sequential data modeling
- Evaluated model performance using metrics such as accuracy, precision, recall, and F1-score

## Impact

- Enabled early detection of anomalies in industrial systems
- Improved reliability and monitoring capabilities
- Demonstrated application of AI in real-world engineering environments
